# -Password-Strength-Checker
## 📊 Overview This project evaluates the strength of a password based on common security rules.   It helps users understand whether their password is weak, moderate, or strong.

## 🚀 Features
- Checks password length  
- Validates presence of:
- Uppercase letters  
- Lowercase letters  
- Digits  
- Special characters
 
 ## 🧠 Concepts Used
- Conditional statements (if-else)
- String operations
- Functions

## 💻 Tech Stack
- Python
